# Responsive Portfolio Website

A multi-page **personal portfolio website** built using **HTML5 and CSS3**.  
This project demonstrates semantic HTML structure, responsive design, and modern CSS styling techniques.

---

##  Features

- Semantic **HTML5 structure**
- **Multi-page website**
- Responsive layout for **desktop and mobile**
- **CSS3 styling** with a single stylesheet
- Hover effects and animations
- HTML5 **form validation**
- Clean and organized code

---

##  Pages Included

###  Home Page
- Introduction section
- Profile image
- Social media links

### About Page
- Personal information
- Career goals
- Semantic HTML elements

###  Project Page
- Blog posts using the `<time>` element
- Project cards with images

###  Contact Page
- HTML5 contact form
- Name, email, and message fields
- Built-in validation
